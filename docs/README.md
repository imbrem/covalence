# docs/ — true documentation

**Contract:** everything in here must be *true* — what the codebase actually is,
kept aggressively in sync with it. Aspirations, plans, and design exploration do
NOT live here (see [`../notes/`](../notes/README.md)).

**Authorship (current policy):** written by the maintainer, not AI-generated —
until the project vision is fully written out by hand. Agents: do not add or
substantially rewrite prose here; flag gaps instead. Exception:
machine-*generated* artifacts, which are true by construction:

- [`deps/`](./deps/) — the crate dependency graph + TCB closure, regenerated by
  `bun run deps` (CI-gated by `bun run deps:check`).
