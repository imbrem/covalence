# Relational residual

- **Term ID:** T0001
- **Status:** Draft

Given relations with compatible endpoints, a right or left relational residual
is an adjoint to composition. In order-enriched notation, the right residual
packages the equivalence

```text
r ; x ≤ s    exactly when    x ≤ r \ s
```

and the left residual packages its mirror image

```text
x ; r ≤ s    exactly when    x ≤ s / r.
```

Thus a residual characterizes the greatest relation satisfying a composition
constraint. In the Covalence logic API this is an optional capability: every
backend can still implement categories and basic relational algebra without
providing residual syntax or the proof transformations witnessing these
adjunctions.

“Optional” describes the capability boundary, not the definition or laws.
